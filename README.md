# Come and drink stuff

- - -

## Installing

Install required packages

```javascript
npm install
```

## Running

Run on local server

``` javascript
yarn start-dev
```

## Building

``` javascript
yarn build-dev
```

## Push to origin

``` javascript
git add .
git commit -m "another commit"
git push origin
```

## Deploying

``` javascript
git add .
git commit -m "another commit"
git push heroku master

heroku open
```
